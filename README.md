# Proptech Landing Page

A premium, AI-powered real estate investment landing page.

## Features
- **Cinematic Design**: Modern aesthetics with smooth animations using Framer Motion.
- **Dark Mode**: Fully functional theme switching support.
- **Market Insights**: Interactive property trend visualizations.
- **Responsive**: Optimized for all devices.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, Lenis (Smooth Scroll)
- **Charts**: Recharts
- **Icons**: Lucide React

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
