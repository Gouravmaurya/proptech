# Project Overview: PropInvest AI / Haven

## Introduction
PropInvest AI (evolving into Haven) is a sophisticated US-focused real estate investment copilot. It allows investors to screen properties, perform detailed financial underwriting (Cap Rate, cash flow, IRR, NPV), and run multi-agent workflows (Scout, Underwriting, Legal, Communication) to assess deals.

## Core Value Proposition
- **Agentic Workflow**: Specialized AI agents handle different aspects of the deal cycle (Scouting, Financial Analysis, Legal Review).
- **Deep Financial Modeling**: Deterministic, math-first financial engine (not LLM-hallucinated) for reliable investment metrics.
- **US Market Focus**: Tailored for US terminology, tax structures, and financing norms (30-year fixed, standard LTVs).
- **Modern Stack**: Built on Next.js 16 (App Router), React 19, and Tailwind CSS.

## Key Features
- **AI Dashboard**: Interactive chat interface to query market data and run property analyses.
- **Multi-Agent Pipeline**: 
  - **Scout**: Finds and filters deals.
  - **Financial**: Runs deep numbers (DSCR, 10-year projections).
  - **Legal/Diligence**: Flags risks in contracts or descriptions.
  - **Communication**: Drafts outreach emails to brokers.
- **Interactive Projections**: 10-year Bear/Base/Bull scenario modeling.
- **Comparison Tools**: Side-by-side deal evaluation.

## Target Audience
- Individual real estate investors.
- Agent teams needing faster underwriting.
- Investment analysts.

## Evolution to "Haven"
The project is being rebranded/evolved to "Haven", with a strong emphasis on:
- **Premium UI/UX**: Shifting from a standard dashboard to a high-end, immersive experience (Glassmorphism, organic shapes, fluid animations).
- **Enhanced Agent Interactivity**: Making the agents feel more alive and collaborative.
